Política de Privacidade – Projeto Acadêmico de Chatbots

Este documento detalha as práticas de privacidade referentes à sua interação com os chatbots desenvolvidos como parte de um Trabalho de Conclusão de Curso (TCC) em Engenharia de Computação na Universidade de Uberaba (UNIUBE).

1. Finalidade do Projeto

Este é um projeto estritamente acadêmico, desenvolvido com ferramentas gratuitas. O objetivo é comparar o desempenho de um chatbot tradicional com um chatbot generativo no atendimento ao cliente. Sua participação e os dados coletados são fundamentais para a análise e os resultados desta pesquisa.

2. Coleta de Dados

Ao interagir com um dos nossos chatbots, as seguintes informações são coletadas:

  • Mensagens do Usuário: O conteúdo das mensagens que você envia via WhatsApp é recebido e processado por nossa aplicação em Python.
  • Histórico da Conversa: Todo o diálogo entre você e o chatbot é registrado em logs para análise posterior.
  • Respostas a Questionários: Ao final da interação, solicitamos que você responda a perguntas sobre sua satisfação e a experiência geral. Suas respostas são armazenadas.
  • Dados de Interação: Informações como horário de início e fim da conversa são registradas para calcular métricas como o tempo médio de atendimento.

3. Uso das Informações

Os dados coletados são utilizados exclusivamente para os seguintes fins acadêmicos:

  • Avaliar a satisfação do usuário com cada tipo de chatbot.
  • Medir a taxa de sucesso e o índice de falha de cada modelo.
  • Analisar a eficiência, como o tempo de resposta.
  • Fornecer dados para a discussão e conclusão da pesquisa.

4. Compartilhamento com Terceiros

Para o funcionamento do experimento, utilizamos as seguintes APIs de terceiros:

  • API do WhatsApp Business (Meta): Utilizada para receber e enviar as mensagens da conversa.
  • API do Google Gemini: No caso do chatbot generativo, o conteúdo da sua mensagem é enviado à API do Google para processamento e geração de uma resposta adequada.

Nenhuma informação será utilizada para fins comerciais ou compartilhada com outras entidades além das necessárias para o funcionamento técnico do projeto.

5. Armazenamento e Segurança

As informações das conversas e as respostas dos questionários podem ser consultadas ou armazenadas em um banco de dados SQL Server. Para este estudo, a aplicação foi executada em um servidor local e controlado.

Ao participar deste experimento, você concorda com a coleta e o uso de seus dados conforme descrito acima, para os fins exclusivos desta pesquisa acadêmica.
